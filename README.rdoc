= Anysee（エニシー）: 人と人とを繋げるコミュニケーションツール

== Anysee（エニシー）とは？
自分と同じ興味を持っている人達と、情報交換や交流の場を提供するツールです。
以下のような機能を持つ予定です。

* つぶやき機能
* コミュニティー機能
* 勉強会機能

== Anysee（エニシー）の由来
Anysee（エニシー）は、人と人との繋がり〜縁（えにし）〜を結びつけるサービスを
作りたいという想いから始まりました。

人と交流を始めるためには、相手のことがよくわからないといけません。
縁（えにし）を結びつけるためには、自分の事を相手に知ってもらい、相手の事も
理解する必要があります。
このツールでお互いの事を理解し、新しい縁（えにし）を結ぶ事ができるよう
縁（えにし）とAny see（なんでもわかる）を繋ぎ合わせ、Anysee（エニシー）と
名付けました。
